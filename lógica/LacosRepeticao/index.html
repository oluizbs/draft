Laços de Repetição

Os laços de Repetição são estruturas utilizadas para repetir um processo por uma determinada condição, por exemplo, imprima "Olá" enquanto o número for menor que 5.
Faça um programa que imprima "Olá" 5 vezes no console. A forma mais ineficiente de se fazer isso seria assim:
console.log("Olá");
console.log("Olá");
console.log("Olá");
console.log("Olá");
console.log("Olá");
Viu? Um mesmo código é repetido diversas vezes, e em grande escala essa maneira não seria nem um pouco eficiente. Para isso existem os laços de repetições, sendo eles, For, While, DoWhile, etc. Usando o For o código poderia ser assim:
For(let i = 0; i < 5; i++){
console.log("Olá");
};
Essa variavel "i" é declarada para "controlar" a condição. Traduzindo o código, o valor de i é 0; enquanto o valor de i for menor que 5; o For irá se repetir, adicionando 1(i++).
Esse "i++" pode ser utilizado de duas formas, nessa forma será mostrado o valor e depois adicionado mais 1, e ++i primeiro será feito a operação e depois será impresso.
